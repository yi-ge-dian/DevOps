主库51

```shell
set global read_only=1;
flush logs;
show master status;
```

从库52

```shell
show slave status\G;

# 查看还有多少进度落后
select  master_post_wait('mysql-bin.000001', 154); --主库日志文件和位置
show processlist;

stop slave;

reset master;
reset slave;

show master status; --记住主库日志文件和位置
```

主库51配置相关参数
```shell
cp /data/3306/etc/my.cnf /data/3306/etc/my.cnf.master
vim /data/3306/etc/my.cnf
# 开启从库相关参数
```

从库52配置相关参数
```shell
cp /data/3306/etc/my.cnf /data/3306/etc/my.cnf.slave
vim /data/3306/etc/my.cnf
# 关闭从库相关参数
```

重启主库51和从库52

```shell
systemctl restart mysqld3306
```

从库51连接主库52

```shell
change master to 
    master_host='10.0.0.52', 
    master_user='repuser', 
    master_password='123456',
    master_log_pos=1;
    GET_SOURCE_PUBLIC_KEY=1;
start slave;
show slave status\G;
```
