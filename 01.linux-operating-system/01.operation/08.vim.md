尾行模式：
```shell
# 显示行号
:set nu
:set nonu 

# 搜索内容xx
/xx 
  n # 往下搜索
  N # 往上搜

# 保存并退出
:wq
# 强制退出不保存
:q

# 只替换光标所在行的第一个单词, a->b
:s/a/b/
# 替换光标所在整行的单词, a->b
:s/a/b/g
# 替换整个文件的第一个单词, a->b
:%s/a/b/
# 替换整个文件的所有单词, a->b
:%s/a/b/g
```

命令模式：
```shell
G   # 快速移动光标到末尾
10G # 快速到第10行
gg  # 快速移动光标到首行
$   # 快速到行尾
^   # 快速到行首

dw # 删除单词
dd # 删除光标所在行
dG # 删除光标到尾行
dgg # 删除光标到第一行
d$ # 删除光标到行尾
d^ # 删除光标到行首
УУ # 复制
5yy # 复制5次
p # 粘贴
5p # 粘贴5次

u # 撤销
```

编辑模式:
```shell
i # 在光标前插入
a # 在光标后插入
o # 在光标下一行插入新行
O # 在光标上一行插入新行
```

可视模式：ctrl + v:
```shell
#批量插入
1. 选中区域，shift + i
2. 输入内容
3. 两次esc
#批量删除
1. 选中区域
2. dw删除
```
