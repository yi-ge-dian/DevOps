查看主从状态

```sql
postgres
=#
SELECT client_addr, state, sent_lsn, write_lsn, flush_lsn, replay_lsn
FROM pg_stat_replication;
client_addr
|   state   |  sent_lsn  | write_lsn  | flush_lsn  | replay_lsn 
---------------+-----------+------------+------------+------------+------------
 172.18.27.199 | streaming | 0/28F8D700 | 0/28F8D700 | 0/28F8D700 | 0/28F8D700
```

- client_addr：连接到主库的从库的 IP 地址。
- state：复制状态，"streaming" 表示正在流式复制。
- sent_lsn：主库已经发送给从库的最新 WAL 日志位置。
- write_lsn：从库已经写入磁盘的最新 WAL 日志位置。
- flush_lsn：从库已经刷新到磁盘的最新 WAL 日志位置。
- replay_lsn：从库已经应用到数据库的最新 WAL 日志位置。

如果想清除数据，应该以 replay_lsn 为准。

查看相应的 wal 文件：

```shell
SELECT pg_walfile_name(' 5/CE49E0B0');
```
