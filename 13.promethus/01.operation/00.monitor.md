监控 node_exporter

修改配置文件

```shell
 vim /data/prometheus/etc/prometheus.yml
 
 - job_name: "node-exporter"
    metrics_path: /metrics
    scheme: http
    static_configs:
      - targets:
        - 10.0.0.72:9100
        - 10.0.0.73:9100
```

检查配置文件是否正确

```shell
/usr/local/prometheus/promtool check config /data/prometheus/etc/prometheus.yml
```

加载配置

```shell
curl -X POST 10.0.0.71:9090/-/reload
```
