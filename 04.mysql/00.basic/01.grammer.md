命令语法
```sql
GRANT
    priv_type [(column_list)]  
      [, priv_type [(column_list)]] ...
    ON [object_type] priv_level  
    TO user_or_role [, user_or_role] ...
    [WITH GRANT OPTION]
    [AS user
        [WITH ROLE
            DEFAULT
          | NONE
          | ALL
          | ALL EXCEPT role [, role ] ...
          | role [, role ] ...
        ]
    ]
}
-- 1. 命令语法中，有大写的信息，相当于命令关键字符
-- 2. 命令语法中，有小写的信息，需要用户自行定义
-- 3. 命令语法中，中括号部分信息，表示可选输入
-- 4. 命令语法中，大括号部分信息，需要选择其一进行使用
```
